测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                            
PA01    电压比较器正端P输入     接直流稳压源  
PA15    VC1 OUT                 接示波器输入   

辅助工具
--------------------------------------------------------------------------------
示波器、直流稳压源

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、连接电压比较器正端P接外部直流稳压源，调整直流电源到ResDivOut以下。
2、打开工程编译并运行。
3、调整直流电源到ResDivOut以上，VC1 OUT输出高电平，LED闪烁一次。
4、调整直流电源到ResDivOut以下，VC1 OUT输出低电平，LED无变化。
5、重复3-4。

================================================================================
注意事项
================================================================================
设置ResDivOut = (23/64)*Vcc ，使用3.3V供电时ResDivOut电压约为1.2V，使用5V供电时
ResDivOut电压约为1.8V
