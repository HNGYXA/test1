测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED    
PA01    电压比较器正端P输入     接直流稳压源  
PA15    VC1 OUT                 接示波器输入         

辅助工具
--------------------------------------------------------------------------------
示波器、直流稳压源 

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行。
2、电压比较器正端P接外部直流稳压源。
3、调整直流电源到1.5V以上，VC1 OUT输出电平翻转。
4、再调整直流电源到1.5V以下，VC1 OUT输出电平翻转，LED闪烁一次。
5、重复3-4。
