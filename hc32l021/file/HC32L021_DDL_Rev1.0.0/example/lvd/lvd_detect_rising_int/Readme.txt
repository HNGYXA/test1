================================================================================
功能描述
================================================================================
本样例主要展示低电压检测器（LVD）的上升沿中断功能

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED           
PA03    LVD输入源               接直流电源
PA06    LVD OUT                 接示波器输入                 

辅助工具
--------------------------------------------------------------------------------
示波器、直流电源

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、连接LVD输入源到直流电源，直流电源输出1.9v以上。
2、打开工程编译并运行。
3、调整直流电源到1.7V以下(因LVD输入源为负端，故此时表现为上升沿触发)。
4、LED亮1s后熄灭，表示触发中断。
5、调整直流电源到1.9V以上，LED无变化。

================================================================================
注意事项
================================================================================
1、测试过程中也可以用示波器同步观测LVD OUT的电平变化（JP2需短接）。
