功能描述
================================================================================
本样例主要展示低电压检测器（LVD）的高电平中断功能

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                            
PA06    LVD OUT                 接示波器输入

辅助工具
--------------------------------------------------------------------------------
示波器、直流电源

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、连接VCC到直流电源，直流电源输出2.5v以上。
2、打开工程编译并运行。
3、调整直流电源到2.4V以下，保持(因VCC为负端，故此时表现为高电平触发)。
4、LED闪烁，表示一直进入中断(若电压过低，LED亮度可能不明显，可用示波器观测端口电平)。
5、调整直流电源到2.5V以上，LED无变化。

================================================================================
注意事项
================================================================================
1、测试过程中也可以用示波器同步观测LVD OUT的电平变化（JP2需短接）
