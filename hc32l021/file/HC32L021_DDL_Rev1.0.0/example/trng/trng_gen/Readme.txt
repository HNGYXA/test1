功能描述
================================================================================
本样例主要展示随机数生成功能。
说明：
本样例上电初次使用初始化功能生成一次64位随机数，程序运行期间再生成一次随机数。
并获得和比较随机数的随机性。

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、运行后查看pu32Data0两次获取的随机数的值，（非全0和全1，且两次生成的随机数不一
    致则功能正确）

================================================================================
注意事项
================================================================================
1、若程序上电后第一次生成随机数，则只需要执行TRNG_Init()即可；
2、在第一次生成随机数之后，如果还需要再次获取随机数，则必须使用TRNG_Generate()
    来生成。
