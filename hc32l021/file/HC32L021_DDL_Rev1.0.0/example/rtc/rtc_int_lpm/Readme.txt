================================================================================
功能描述
================================================================================
本样例主要涉及时钟模块唤醒中断实例

说明：
RTC配置5s唤醒中断
中断中翻转LED
主函数等待按键按下进入低功耗模式

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------                         
PA00    LED 
PA12    USER KEY

辅助工具
--------------------------------------------------------------------------------
无

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、重新编译代码运行。
2、观察LED 5s翻转一次。
3、按下USER KEY，MCU将进入深度睡眠模式。
4、在深度睡眠模式下LED的输出依旧。

================================================================================
注意事项
================================================================================
1、芯片进入深度睡眠模式后，可将电流表串入Pwr Test的两个PIN脚测试MCU的功耗。
