测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA01    SPI_MISO               对应主机的SPI_MISO       
PA02    SPI_MOSI               对应主机的SPI_MOSI 
PA03    SPI_SCK                对应主机的SPI_SCK  
PA04    SPI_CS                 对应主机的SPI_CS   

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、与spi_master_poll样例联合使用
2、准备测试环境，打开样例工程编译、下载后运行
3、下载程序并先于主机全速运行，等待与主机通信
4、可观察主机的LED灯，如果主机接收和发送的数据一致，则主机板子上的LED灯会闪烁；
   也可通过调试窗口，观察从机的数据接收状态，查看u16SlaveRxBuf和u16SlaveTxBuf数组的
   数据是否一致。
