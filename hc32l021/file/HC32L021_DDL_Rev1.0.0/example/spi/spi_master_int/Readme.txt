测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY                
PA01    SPI_MISO               对应从机的SPI_MISO       
PA02    SPI_MOSI               对应从机的SPI_MOSI 
PA03    SPI_SCK                对应从机的SPI_SCK  
PA04    SPI_CS                 对应从机的SPI_CS

--------------------------------------------------------------------------------
无

--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、与spi_slave_int样例联合使用
2、准备测试环境，打开样例工程编译、下载后运行
3、先运行从机程序，再运行主机程序，按下按键后，观察主机的LED灯，如果主机发送和
   接收的数据一致，则板子上的LED灯会闪烁；也可通过调试窗口，
   查看u16MasterTxBuf和u16MasterRxBuf数组的数据是否一致。
