功能描述
================================================================================
本样例主要展示从机SPI作为从机的中断收发功能

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY                
PA01    SPI_MISO               对应主机的SPI_MISO       
PA02    SPI_MOSI               对应主机的SPI_MOSI 
PA03    SPI_SCK                对应主机的SPI_SCK  
PA04    SPI_CS                 对应主机的SPI_CS   

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、与spi_master_int样例联合使用
2、准备测试环境，打开样例工程编译、下载后运行
3、下载程序并先于主机全速运行，等待与主机通信
4、观察从机的LED灯，如果从机接收和发送的数据一致，则板子上的LED灯会闪烁；也可通过调试窗口，
   查看u16SlaveRxBuf和u16SlaveTxBuf数组的数据是否一致。
