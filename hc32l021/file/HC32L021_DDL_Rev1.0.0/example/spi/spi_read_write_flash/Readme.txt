功能描述
================================================================================
本样例主要展示使用SPI读写W25Qxx FLASH

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY                
PA01    SPI_MISO               对应从机的SPI_MISO       
PA02    SPI_MOSI               对应从机的SPI_MOSI 
PA03    SPI_SCK                对应从机的SPI_SCK  
PA04    SPI_CS                 对应从机的SPI_CS   

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、按下按键，等待数据读写完成，当写入的数据和读出的数据一致时，板子上的LED灯会闪烁；
   也可通过调试窗口，查看u16WriteData和u16ReadData数组的数据是否一致。
