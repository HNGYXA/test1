功能描述

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA12    USER KEY                
PA02    SPI_MOSI               对应从机的SPI_MOSI 
PA03    SPI_SCK                对应从机的SPI_SCK  
PA04    SPI_CS                 对应从机的SPI_CS   

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、与spi_slave_receive_simplex样例联合使用
2、准备测试环境，打开样例工程编译、下载后运行
3、先运行从机程序，再运行主机程序，待从机按键按下后，按下主机按键，与从机通信，观察从机
   的LED灯，如果主机发送和从机接收接收的数据一致，则从机板子上的LED灯会闪烁

================================================================================
注意事项
================================================================================ 
1、请保证通信顺序，从机按键按下后，等待主机按下按键进行通信
2、传输速率高的时候，主机可选择将采样方式设置成延时采样
