功能描述
================================================================================
本样例主要展示从机SPI单线半双工收发功能

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA12    USER KEY     
PA01    SPI_MISO               对应主机的SPI_MOSI 
PA03    SPI_SCK                对应主机的SPI_SCK  
PA04    SPI_CS                 对应主机的SPI_CS   

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、与spi_master_poll_halfduplex样例联合使用
2、准备测试环境，打开样例工程编译、下载后运行
3、下载程序并先于主机全速运行，按下从机按键，等待与主机通信
4、可观察主机的LED灯，如果主机接收和发送的数据一致，则主机板子上的LED灯会闪烁；
   也可通过调试窗口，观察从机的数据接收状态，查看u16SlaveRxBuf和u16MasterTxBuf数组
   的数据是否一致。

================================================================================
注意事项
================================================================================ 
1、在当前传输速率下，传输数据线间串入1K限流电阻
2、请保证通信顺序，从机按键按下后，等待主机按下按键进行通信
3、传输速率高的时候，主机可选择将采样方式设置成延时采样
