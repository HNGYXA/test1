================================================================================
功能描述
================================================================================
本样例展示HSI2C从机传送功能（中断方式）

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0（作为从机）
SK_L021_TSSOP20_Rev1.0（作为主机）

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY                
PA10    SDA                    主机硬件板的SDA、接示波器输入
PA11    SCL                    主机硬件板的SCL、接示波器输入

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，需要另一块SK_L021_TSSOP20_Rev1.0硬件板作为主机
2、打开本样例工程，编译、下载后运行
3、打开主机样例，hsi2c_mcu_master_poll或者hsi2c_mcu_master_int样例工程编译、下载后运行
4、当LED亮起后，查看主机发送的数据u8WriteData[]和从机的接收数据u8ReadData[]一致;
   主机接收的数据u8ReadData[]和从机的发送数据u8WriteData[]一致;
   也可通过示波器观察发送和接收的数据等。                

================================================================================
注意事项
================================================================================
1、需准备主机硬件板配合测试
2、需先运行从机，再运行主机
3、需根据主机发送协议设置stcHsi2cSlaveInit.u8SubAddrSize的值
   当该值!=0时：接收的Subaddress值存放在stcHsi2cCom.u32SubAddr中
   当该值=0时：接收的Subaddress值存放在发送和接收数组中
4、测试用板上跳帽JP6需断开
5、需根据实际应用场景，为SCL和SDA外接合适阻值的上拉电阻
