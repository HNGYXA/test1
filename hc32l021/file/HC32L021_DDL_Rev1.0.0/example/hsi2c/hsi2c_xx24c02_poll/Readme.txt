================================================================================
功能描述
================================================================================
本样例展示HSI2C主机写、读EEPROM（轮询方式）

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY                
PA10    SDA                    接示波器输入
PA11    SCL                    接示波器输入

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、打开本样例工程，编译、下载后运行
2、当LED亮起后，查看主机发送数据u8WriteData[]和接收数据u8ReadData[]一致;
   也可通过示波器观察发送和接收的数据等。

================================================================================
注意事项
================================================================================
1、测试用板上跳帽JP6需连接
2、需根据实际应用场景，为SCL和SDA外接合适阻值的上拉电阻
