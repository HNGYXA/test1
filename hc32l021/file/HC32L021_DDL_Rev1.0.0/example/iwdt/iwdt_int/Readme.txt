功能描述
================================================================================
本样例主要展示独立看门狗的中断及喂狗功能。

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED
PA12    USER KEY                        

辅助工具
--------------------------------------------------------------------------------

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、打开工程并重新编译、运行。
2、如果在看门狗溢出前(约3.2s内)按下按键(喂狗)则LED不会点亮(溢出),
   否则LED被点亮(进入溢出中断)。
