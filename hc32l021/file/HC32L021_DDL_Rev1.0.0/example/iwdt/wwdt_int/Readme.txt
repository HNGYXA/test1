功能描述
================================================================================
本样例主要展示窗口看门狗的中断及喂狗功能。

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED
PA12    USER KEY                        

辅助工具
--------------------------------------------------------------------------------

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、打开工程并重新编译、运行。
2、如果在窗口时间内(3.2s~6.4s)按下按键喂狗，则不会发生看门狗中断(LED不亮);
 如果在窗口时间外(3.2s以内)按下按键喂狗(LED闪烁一次)，
 或溢出前(6.4s)没有喂狗，则发生中断(LED点亮)。
