功能描述
================================================================================
本样例主要展示窗口看门狗的复位功能。

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED
PA12    USER KEY                    

辅助工具
--------------------------------------------------------------------------------

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、打开工程并重新编译、运行。
2、如果在窗口时间内按下按键喂狗(3.2s~6.4s)，则不会发生看门狗复位(LED不亮);
 如果在窗口时间外按下按键喂狗(3.2s以内)，或溢出前没有喂狗，则发生复位(LED闪烁)。
