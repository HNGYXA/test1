功能描述
================================================================================
本样例主要展示独立看门狗的低功耗状态暂停功能。

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED
PA12    USER KEY                        

辅助工具
--------------------------------------------------------------------------------
无

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、打开工程并重新编译、运行。
2、若未按下按键，则执行看门狗中断功能，溢出时(约3.2s)LED闪烁；
 若按下按键，则进入低功耗深度睡眠模式，看门狗自动暂停运行，LED不会闪烁。

================================================================================
注意事项
================================================================================
1、芯片进入深度睡眠模式后，可将电流表串入Pwr Test的两个PIN脚测试MCU的功耗。
