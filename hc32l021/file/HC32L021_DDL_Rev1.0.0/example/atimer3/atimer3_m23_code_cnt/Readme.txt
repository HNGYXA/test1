功能描述
================================================================================
本样例展示Atimer3的三角波作为载波的独立PWM输出功能

================================================================================
测试环境
===============================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA11    TIM3 CH0A               接PA03 
PA01    TIM3 CH0B               接PA04
PA03    模拟正交信号A           接PA11
PA04    模拟正交信号B           接PA01
PA00    LED                     接示波器输入 
 
辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无
