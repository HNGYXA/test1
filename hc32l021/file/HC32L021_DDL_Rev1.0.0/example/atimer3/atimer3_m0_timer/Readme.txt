功能描述
================================================================================
本样例主要展示Timer3的定时功能。
说明：
本样例使用定时器定时，定时溢出后产生中断，并重新载入初值进行定时，并再次溢出进入中断。

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、重载模式下，TIM3从ARR计数到溢出，会产生溢出中断，timer中断周期100ms
3、在中断内对PA00输出进行翻转，控制LED灯亮灭
