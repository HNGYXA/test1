功能描述
================================================================================
本样例展示Atimer3的三角波作为载波的独立PWM输出功能

================================================================================
测试环境
===============================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA11    TIM3 CH0A               接示波器输入 
PA01    TIM3 CH0B               接示波器输入 
PA02    TIM3 CH1A               接示波器输入 
PA03    TIM3 CH1B               接示波器输入 
PA04    TIM3 CH2A               接示波器输入 
PA05    TIM3 CH2B               接示波器输入 

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、观察波形输出（六路PWM）
3、设置间隔计数为0,OV Mask使能，每个三角波周期下溢触发中断，并且改变输出占空比

================================================================================
注意事项
================================================================================
1）如果要测试CHA与CHB的互补PWM功能（单点比较：CHA占空比由CCR0A设置，CHB与CHA互补）
   请将stcAtim3BaseConfig.u32PWMTypeSelect  = ATIM3_M23_M23CR_COMP_PWM_INDEPT; 
   改为stcAtim3BaseConfig.u32PWMTypeSelect  = ATIM3_M23_M23CR_COMP_PWM_COMP;
