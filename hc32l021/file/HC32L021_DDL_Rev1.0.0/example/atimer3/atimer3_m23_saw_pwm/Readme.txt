功能描述
================================================================================
本样例展示Timer3的锯齿波作为载波的独立PWM输出功能

--------------------------------------------------------------------------------

PA01    ATIM3 CH0B               接示波器输入 
PA02    ATIM3 CH1A               接示波器输入 
PA03    ATIM3 CH1B               接示波器输入 
PA04    ATIM3 CH2A               接示波器输入 
PA05    ATIM3 CH2B               接示波器输入

无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、观察波形输出（六路PWM）
3、设置间隔计数为0，每个周期Uev触发中断，并且改变输出占空比
