功能描述
================================================================================
本样例展示ATIM3的带硬件死区互补PWM输出功能，并触发ADC的采样

================================================================================
测试环境
===============================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA11    TIM3 CH0A               接示波器输入 
PA01    TIM3 CH0B               接示波器输入 
PA02    TIM3 CH1A               接示波器输入 
PA03    TIM3 CH1B               接示波器输入 
PA04    TIM3 CH2A               接示波器输入 
PA05    TIM3 CH2B               接示波器输入 
PA09    ADC转换通道0输入         接待转换电压源

辅助工具
--------------------------------------------------------------------------------
示波器
外部输入电压源源

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、观察波形输出（六路两两对称的互补PWM，并且带有死区时间），LED翻转输出
3、设置间隔计数为0,OV Mask使能，每个周期下溢Uev触发中断，并触发ADC采样
4、中断可读取u32AdcResult9转换结果
