================================================================================
功能描述
================================================================================
本样例展示系统时钟输出功能

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY               
PA14    时钟频率输出            接示波器输入

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、按下USER KEY，PA14输出频率，因系统时钟为48MHz，HCLK为1分频值，PA14输出8分频，
   对应时钟频率为6MHz

================================================================================
注意事项
================================================================================
1、本样例使用PA14（SWCLK）输出时钟频率，当PA14切换功能为时钟频率输出后，SWD功能将不可用
2、当需重新下载程序时，需在复位后2s内进行，否则会因SWD功能不可用而导致下载失败
