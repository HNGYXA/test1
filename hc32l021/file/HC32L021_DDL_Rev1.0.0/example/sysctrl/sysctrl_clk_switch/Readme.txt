================================================================================
功能描述
================================================================================
本样例展示系统时钟切换功能

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY                
PA14    时钟频率输出            接示波器输入
PA11    EXTH时钟                输入8MHz时钟

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、每按下USER KEY，系统时钟频率切换，依次为6MHz（RC48M）、
   32MHz（RC48M）、48MHz（RC48M）、4MHz（RC48M）、8M（EXTH）、32768Hz（RCL）、
   38400Hz（RCL）、32768Hz（XTL）、4MHz（RC48M）;
   因PA14设置为HCLK的8分频值，示波器显示时钟频率值为上述值的8分频值，依次为：
   750K、4M、6M、500K、1M、4K、4.8K、4K、500K。

================================================================================
注意事项
================================================================================
1、本样例使用PA14（SWCLK）输出时钟频率，当PA14切换功能为时钟频率输出后，SWD功能将不可用
2、当需重新下载程序时，需在复位后2s内进行，否则会因SWD功能不可用而导致下载失败
