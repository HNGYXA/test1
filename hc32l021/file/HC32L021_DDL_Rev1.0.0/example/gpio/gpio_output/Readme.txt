--------------------------------------------------------------------------------

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、PA00每一秒输出翻转一次，可观测到PA00的波形每隔1s点翻转，LED每隔1s闪烁一次
