功能描述
================================================================================
本样例主要展示GPIO输入功能。
说明：
本样例使用USER KEY作为输入功能演示，并根据输入的值设置LED的状态。

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY                

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、按下USER KEY，LED闪烁；释放USER KEY，LED关闭。
