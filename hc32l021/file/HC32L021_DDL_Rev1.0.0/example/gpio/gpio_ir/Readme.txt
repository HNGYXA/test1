功能描述
================================================================================
本样例主要展示GPIO的红外输出功能。
说明：
本样例使用端口复用功能，输出38K红外信号，红外输出极性选择为反向输出。

--------------------------------------------------------------------------------

示波器

无

使用步骤

2、示波器探头连接PA08可查看红外输出信号
