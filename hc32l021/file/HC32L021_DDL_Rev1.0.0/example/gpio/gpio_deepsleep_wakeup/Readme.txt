功能描述
================================================================================
本样例主要展示GPIO外部中断唤醒深度休眠功能。
说明：
本样例使用USER KEY作为GPIO外部中断通过下降沿触发唤醒深度休眠，唤醒后进入中断服务程序，将
LED点亮延时2s,执行完后再次进入深度休眠模式。

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY                

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、重新上电,2s钟后进入深度休眠模式，（此时也可通过测量芯片功耗来确认进入深度休眠模式）
3、按下USER KEY，中断唤醒深度休眠模式，LED点亮并维持2s,（此2s期间也可通过测量芯片功
   耗来确认进入运行模式）
4、LED点亮2s后芯片再次进入深度休眠模式

================================================================================
注意事项
================================================================================
1、该Demo主要演示中断唤醒休眠模式功能，该Demo基于配套STK板将空置的GPIO均配置为模拟，
   已最大限度降低功耗。
2、芯片进入深度休眠模式后，可将电流表串入Pwr Test的两个PIN脚测试MCU的功耗。
