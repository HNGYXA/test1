功能描述
================================================================================
本样例主要展示GPIO外部中断触发功能。
说明：
本样例使用USER KEY作为GPIO外部中断通过下降沿触发中断，进入中断服务程序，将
LED点亮延时2s。

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY                

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、按下USER KEY，触发外部中断，LED点亮并维持2s；
3、LED点亮2s后熄灭，程序正常运行。

================================================================================
注意事项
================================================================================
1、该样例主要演示中断下降沿触发功能，其他方式的触发只需修改中断触发方式配置即可。
