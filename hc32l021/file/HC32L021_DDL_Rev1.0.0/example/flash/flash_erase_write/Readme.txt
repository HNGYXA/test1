================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载
2、在main函数中的8位写后设置断点，运行程序
3、程序到断点处停止，打开Memory窗口，地址0xFE00到地址0xFE7F区域的数据和u8TestData[]相同
4、在main函数中的16位写后设置断点，运行程序
5、程序到断点处停止，打开Memory窗口，地址0xFE00到地址0xFEFF区域的数据和u16TestData[]相同
6、在main函数中的32位写后设置断点，运行程序
7、程序到断点处停止，打开Memory窗口，地址0xFE00到地址0xFFFF区域的数据和u32TestData[]相同
