功能描述
================================================================================
本样例主要系统时钟校准功能。
说明：
本样例使用外部1M时钟对内部RCL进行32768Hz校准。

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY                
PA14    时钟频率输出            接示波器输入
PA11    CTRIM_ETRTOG           信号发生器（1M精准时钟）

辅助工具
--------------------------------------------------------------------------------
示波器、信号发生器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、需等待2s切换PA14从swd为gpio模式，避免无法下载程序，在此等待时间内，PA14输出的是
   未校准的RCL的时钟频率
3、按下USER KEY后开始执行校准流程
4、当指示灯亮起，校准成功，PA14输出校准后的RCL的时钟频率，
   自动CTRIM结果CTRIM->TVAL寄存器的值被写入SYSCTRL->RCL_CR的TRIM寄存器

================================================================================
注意事项
================================================================================
1、低速时钟校准，校准时间可能会略微长一点
2、本样例使用PA14（SWCLK）输出时钟频率，当PA14切换功能为时钟频率输出后，SWD功能将不可用
3、当需重新下载程序时，需在复位后2s内进行，否则会因SWD功能不可用而导致下载失败
