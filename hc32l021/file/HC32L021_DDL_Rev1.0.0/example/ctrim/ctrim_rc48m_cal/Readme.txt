功能描述
================================================================================
本样例主要系统时钟校准功能。
说明：
本样例为了操作方便，使用XTL 32768Hz晶振把内部RC48M校准到4MHz。
PA14输出频率，通过示波器观察RC48M_4M频率。

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY                
PA14    时钟频率输出            接示波器输入

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、需等待2s切换PA14从swd为gpio模式，避免无法下载程序，在此等待时间内，PA14输出的是
   未校准的RC48M_4M的时钟频率；
3、按下USER KEY后开始执行校准流程；
4、当指示灯亮起，校准成功，PA14输出校准后的RC48M_4M的时钟频率，
   自动Trim结果CTRIM->TVAL寄存器的值写入SYSCTRL->RC48M_CR的TRIMF与TRIMC寄存器。

================================================================================
注意事项
================================================================================
1、该样例为了操作方便，选用了XTL 32768Hz晶振作为低频时钟信号。
2、如果需要把RC48M校准到其它的值，可根据需要选择其它时钟并修改样例实现。
3、本样例使用PA14（SWCLK）输出时钟频率，当PA14切换功能为时钟频率输出后，SWD功能将不可用
4、当需重新下载程序时，需在复位后2s内进行，否则会因SWD功能不可用而导致下载失败
