测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚  作用          测试连接
--------------------------------------------------------------------------------
PA01：TX  <-------> PC RX
PA00: LED                     
PA12: USER KEY                


辅助工具
--------------------------------------------------------------------------------
无

辅助软件
--------------------------------------------------------------------------------
串口调试工具

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载
2、串口调试工具配置波特率9600bps、偶校验、1stop、数据长度8位
3、重新上电后，窗口显示MCU相关信息；
4、默认状态或释放USERKEY，LED灯慢速闪烁；
5、按下USERKEY，LED快速闪烁；
