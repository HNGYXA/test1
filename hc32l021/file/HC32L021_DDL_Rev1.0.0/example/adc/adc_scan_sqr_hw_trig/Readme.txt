测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA02    ADC转换通道0输入        待转换电压源
PA05    ADC转换通道1输入        待转换电压源
PA03    ADC触发源               接USER KEY（PA12）
PA12    USER KEY

辅助工具
--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、每按一次USER 按键，PA03产生下降沿中断，触发一次ADC转换
3、在主循环里面等待转换完成，读取(u32AdcResult2、u32AdcResult5)
