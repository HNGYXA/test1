功能描述
================================================================================
本样例主要展示ADC软件触发采样，使用查询方式读取转换结果

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA02    ADC转换通道0输入         待转换电压源
PA03    ADC转换通道1输入         待转换电压源
PA05    ADC转换通道2输入         待转换电压源

辅助工具
--------------------------------------------------------------------------------
外部输入电压源

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、每2秒软件自动触发一次ADC转换
3、中断可读取u32AdcResult2、u32AdcResult3、u32AdcResult5转换结果
