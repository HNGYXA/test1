功能描述
================================================================================
本样例主要展示LPUART中断触发ADC采样

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA11    TX  <------------------> PC RX
PA10    RX  <------------------> PC TX
PA02    ADC转换通道0输入         待转换电压源
PA03    ADC转换通道1输入         待转换电压源
PA05    ADC转换通道2输入         待转换电压源

辅助工具
--------------------------------------------------------------------------------
外部输入电压源

辅助软件
--------------------------------------------------------------------------------
串口调试工具

================================================================================
使用步骤
================================================================================
1、准备测试环境，串口调试工具配置波特率9600bps、偶校验、1stop、数据长度8位 
2、打开样例工程编译、下载后运行，PC收到0x55、0xAA说明通信成功
3、PC通过串口调试工具向MCU发送数据，MCU接收后返回
4、LPUART中断触发ADC转换，可debug读取(u32AdcResult2、u32AdcResult3、u32AdcResult5)的转换结果
