功能描述
================================================================================
本样例主要展示GTIM的单次触发计数功能。
说明：
本样例展示了GTIM0的单次触发计数功能。外部输入引脚ETR作为触发输入引脚，输入一次
下降沿触发，GTIM0启动计数一次，产生触发中断，在触发中断点亮LED，GTIM0计数溢出后，
产生溢出中断，在溢出中断关闭LED

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     接示波器输入 
PA05    ETR信号输入             （可外接输入高低电平）   

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、ETR(PA05)输入下降沿，GTIM0启动计数一次，产生触发中断,LED(PA00)点亮，GTIM0计数溢出后，
   产生溢出中断，LED(PA00)关闭

================================================================================
注意事项
================================================================================
1、在触发模式，ETR输入滤波需要先于工作模式设置。
2、BTIM0/1/2与GTIM0不可同时使用，通过SYSCTRL1.CTIMER0_FUN_SEL来控制
3、BTIM3/4/5与GTIM1不可同时使用，通过SYSCTRL1.CTIMER1_FUN_SEL来控制
