功能描述
================================================================================
本样例主要展示BTIM的门控功能。
说明：
本样例展示BTIM1在门控模式下，定时器运行之后，ETR作为门控引脚：
当ETR输入为低的时候，BTIM1运行，溢出后产生中断，并且在中断内翻转LED引脚电平，TOG输出; 
当ETR输入为高的时候，BTIM1停止计数，LED不闪烁，TOG停止翻转。

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、PA05输入高电平，TOG翻转输出停止，LED不闪烁
3、PA05输入低电平，TOG翻转输出，LED闪烁

================================================================================
注意事项
================================================================================
1、测试用板上跳帽JP1需断开
2、BTIM0/1/2与GTIM0不可同时使用，通过SYSCTRL1.CTIMER0_FUN_SEL来控制
3、BTIM3/4/5与GTIM1不可同时使用，通过SYSCTRL1.CTIMER1_FUN_SEL来控制
4、CR.TOGEN不使能的时候，TOG和TOGN输出低电平
   CR.TOGEN使能的时候，TOG和TOGN输出频率相同，相位相反的波形
5、BTIM0_TOG和TOGN即为GTIM0_TOG和GTIM0_TOGN
   BTIM1_TOG和TOGN即为GTIM0_CH0和GTIM0_CH1
   BTIM2_TOG和TOGN即为GTIM0_CH2和GTIM0_CH3
6、BTIM3_TOG和TOGN即为GTIM1_TOG和GTIM1_TOGN
   BTIM4_TOG和TOGN即为GTIM1_CH0和GTIM1_CH1
   BTIM5_TOG和TOGN即为GTIM1_CH2和GTIM1_CH3
7、BTIM0/1/2的ETR引脚即为CTIM0_ETR引脚, BTIM3/4/5的ETR引脚即为CTIM1_ETR引脚
8、BTIM0/1/2的中断向量即为GTIM0中断向量(中断函数命名不同)
9、BTIM3/4/5的中断向量即为GTIM1中断向量(中断函数命名不同)
10、BTIM0/1/2的外设时钟与GTIM0的外设时钟为同一个
11、BTIM3/4/5的外设时钟与GTIM1的外设时钟为同一个
