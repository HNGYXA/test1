功能描述
================================================================================
本样例主要展示GTIM的捕获功能。
说明：
本样例主要展示GTIM0的捕获功能，输入脉冲为上升沿的时候，捕获寄存器的值到CCR0寄存器
待下一次上升沿到来的时候，再捕获一次。计算两次上升沿直接的计数值(舍弃明显错误的值)

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA12    USER KEY                
PA02    外部时钟源信号输入       

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、程序会等待USER KEY按下，在USER KEY按下MCU会启动定时器，进入捕获功能
3、外部时钟源信号输入信号，在调试窗口查看捕获值，LED会闪烁

================================================================================
注意事项
================================================================================
1、BTIM0/1/2与GTIM0不可同时使用，通过SYSCTRL1.CTIMER0_FUN_SEL来控制
2、BTIM3/4/5与GTIM1不可同时使用，通过SYSCTRL1.CTIMER1_FUN_SEL来控制
