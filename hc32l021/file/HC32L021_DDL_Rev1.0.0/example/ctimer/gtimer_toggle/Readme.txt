功能描述
================================================================================
本样例主要展示GTIM的TOG翻转输出功能。
说明：
本样例主要展示GTIM0在PCLK为时钟下，计数溢出，产生溢出中断，并在溢出时候翻转TOG
和TOGN输出一对反相的脉冲波形
================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     
PA03    TOG输出                接示波器输入    
PA04    TOGN输出               接示波器输入

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、用示波器观察PA03和PA04输出的波形，大小相等，相位相反
3、PA00（LED）出现高低电平变换

================================================================================
注意事项
================================================================================
1、BTIM0/1/2与GTIM0不可同时使用，通过SYSCTRL1.CTIMER0_FUN_SEL来控制
2、BTIM3/4/5与GTIM1不可同时使用，通过SYSCTRL1.CTIMER1_FUN_SEL来控制
