功能描述
================================================================================
本样例主要展示GTIM的外部ETR输入计数功能。
说明：
本样例对展示GTIM0使用ETR外部输入脉冲作为计数时钟，计数到周期ARR然后溢出产生溢出中断的
功能。

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                    接示波器输入 
PA05    外部输入脉冲信号        接入高低电平信号       

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、给PA05(CTIM0_ETR)输入高低电平信号
3、GTIM计数到ARR后溢出，产生溢出中断，并翻转LED(PA00)

================================================================================
注意事项
================================================================================
1、测试用板上跳帽JP1需断开
