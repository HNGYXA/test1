功能描述
================================================================================
本样例主要展示BTIM的TOG翻转功能。
说明：
本样例主要展示BTIM1的TOG输出翻转功能，使能TOGGLE功能和TOG和TOGN引脚，
BTIM1计数溢出后，产生溢出中断，并且翻转TOG电平

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     接示波器输入 
PA12    USER KEY                
PA03    TOG输出                 接示波器输入
PA04    TOGN输出                接示波器输入

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、按下USER KEY，PA03和PA04输出频率相同，相位相反的波形，LED闪烁。

================================================================================
注意事项
================================================================================
1、BTIM0/1/2与GTIM0不可同时使用，通过SYSCTRL1.CTIMER0_FUN_SEL来控制
2、BTIM3/4/5与GTIM1不可同时使用，通过SYSCTRL1.CTIMER1_FUN_SEL来控制
3、CR.TOGEN不使能的时候，TOG和TOGN输出低电平
   CR.TOGEN使能的时候，TOG和TOGN输出频率相同，相位相反的波形
4、BTIM0_TOG和TOGN即为GTIM0_TOG和GTIM0_TOGN
   BTIM1_TOG和TOGN即为GTIM0_CH0和GTIM0_CH1
   BTIM2_TOG和TOGN即为GTIM0_CH2和GTIM0_CH3
5、BTIM3_TOG和TOGN即为GTIM1_TOG和GTIM1_TOGN
   BTIM4_TOG和TOGN即为GTIM1_CH0和GTIM1_CH1
   BTIM5_TOG和TOGN即为GTIM1_CH2和GTIM1_CH3
6、BTIM0/1/2的ETR引脚即为CTIM0_ETR引脚, BTIM3/4/5的ETR引脚即为CTIM1_ETR引脚
7、BTIM0/1/2的中断向量即为GTIM0中断向量(中断函数命名不同)
8、BTIM3/4/5的中断向量即为GTIM1中断向量(中断函数命名不同)
9、BTIM0/1/2的外设时钟与GTIM0的外设时钟为同一个
10、BTIM3/4/5的外设时钟与GTIM1的外设时钟为同一个
