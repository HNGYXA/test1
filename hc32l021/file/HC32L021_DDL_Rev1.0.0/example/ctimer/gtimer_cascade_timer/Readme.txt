功能描述
================================================================================
本样例主要展示GTIM的级联功能。
说明：
本样例展示了GTIM0的级联功能，ATIM3的UEV溢出信号输出给GTIM0，作为GTIM0的外部输入
计数时钟，组成一个32bit的定时器。

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA00    LED                     

辅助工具
--------------------------------------------------------------------------------
示波器

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、GTIM0计数到ARR后溢出，产生溢出中断，并在中断内翻转LED(PA00)的电平。 

================================================================================
注意事项
================================================================================
1、BTIM0/1/2与GTIM0不可同时使用，通过SYSCTRL1.CTIMER0_FUN_SEL来控制
2、BTIM3/4/5与GTIM1不可同时使用，通过SYSCTRL1.CTIMER1_FUN_SEL来控制
