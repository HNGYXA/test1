功能描述
================================================================================
本样例主要展示GTIM的门控功能。
说明：
本样例展示了GTIM0的门控功能，外部ETR输入引脚作为门控引脚。
在ETR为低电平时候门控有效，GTIM计数，计数溢出产生溢出中断，翻转TOG和TOGN，
并且在溢出中断内翻转LED引脚
在ETR为高电平时候门控无效，GTIM停止计数。

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、PA14为GTIM0的ETR引脚，
   PA14接高电平，GTIM0停止计数，
   PA14接低电平，GTIM0计数，会翻转PA03和PA04电平，并且计数到ARR后，产生溢出中断，
   在中断内翻转PA00引脚电平，LED快速闪烁。

================================================================================
注意事项
================================================================================
1、测试用板上跳帽JP1需断开
2、BTIM0/1/2与GTIM0不可同时使用，通过SYSCTRL1.CTIMER0_FUN_SEL来控制
3、BTIM3/4/5与GTIM1不可同时使用，通过SYSCTRL1.CTIMER1_FUN_SEL来控制
4、本样例使用PA14（SWCLK）输出时钟频率，当PA14切换功能为时钟频率输出后，SWD功能将不可用
5、当需重新下载程序时，需在复位后2s内进行，否则会因SWD功能不可用而导致下载失败
