功能描述
================================================================================
本样例主要展示串口从机硬件流控方式收发数据

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
从机                            主机
PA01    TX  <-----------------> PA02 RX
PA02    RX  <-----------------> PA01 TX
        GND <-----------------> GND

辅助工具
--------------------------------------------------------------------------------
无

辅助软件
--------------------------------------------------------------------------------
无

================================================================================
使用步骤
================================================================================
1、准备测试环境，与例程lpuart_master联调，分别下载主从机代码
2、使用debug模式，先从机上电，再主机上电
3、在调试窗口观察数组u8SlaveRxData的值是否是0x00~0x09，如果是表示通信正常
