功能描述
================================================================================
本样例主要展示低功耗串口中断收发数据样例

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用           测试连接
--------------------------------------------------------------------------------
PA01    TX  <----------> PC RX
PA02    RX  <----------> PC TX
PA12    USER KEY

辅助工具
--------------------------------------------------------------------------------
无

辅助软件
--------------------------------------------------------------------------------
串口调试工具

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、串口调试工具配置波特率9600bps、偶校验、1stop、数据长度8位
3、代码运行后，PC收到0x55,0xAA表示通信成功，
4、按下按键(PA12)，系统进入深度休眠
5、PC串口发送1字节数据0xXX，唤醒系统，MCU返回0xXX
